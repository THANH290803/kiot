# Setup Guide

## Mục tiêu

Tài liệu setup giúp chạy frontend đúng với backend hiện tại.

## Tài liệu con

- `local-development.md`: chạy local full stack
- `deployment.md`: build/deploy và checklist sau deploy

## Checklist nhanh

1. Cài dependencies workspace.
2. Cấu hình env frontend (`NEXT_PUBLIC_API_URL`, ...).
3. Đảm bảo backend API đang chạy và truy cập được.
4. Chạy `npm run dev --workspace=frontend`.
