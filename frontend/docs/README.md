# Frontend Docs

Tai lieu frontend duoc chia theo pham vi:

- `architecture/`: kien truc tong the, routing, boundaries
- `features/`: tai lieu theo domain `admin` va `user`
- `shared/`: tai lieu cho shared layer, ui, hooks, providers
- `api/`: mapping frontend sang backend API
- `conventions/`: quy uoc coding, naming, import, folder structure
- `setup/`: cach chay local, env, build, deploy
